
# Recurrence Relation for Tower of Hanoi Problem


#### The recurrence relation capturing the optimal execution time of the Towers of Hanoi problem with n discs is :

## Options

T(n) = 2T(n − 2) + 2

T(n) = 2T(n − 1) + n

T(n) = 2T(n/2) + 1

T(n) = 2T(n − 1) + 1

## SOLUTION:
T(n) = 2T(n − 1) + 1
